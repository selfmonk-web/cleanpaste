CleanPaste is a lightweight Chrome extension that cleans copied text in one click.

It removes:
- HTML tags
- Formatting
- Weird spacing
- Unnecessary line breaks

## How it works
1. Copy any text
2. Click the CleanPaste extension
3. Paste clean text anywhere

## Features
- One-click clean
- No ads
- No tracking
- No analytics
- No login required

## Permissions
CleanPaste uses clipboard permissions only to read and write copied text when the user clicks the button.

## Privacy
CleanPaste does not collect, store, or transmit any personal data.
All processing happens locally in the browser.
